### Первое Задание

### Второе задание

Есть несколько выражений

```
    "" + 2 + 0
    "" - 2 + 0
    "4" > "22"
    4 + 7 + "px"
    "3px" - 5
    "30px" + "50px"
    "50px" - "30px" 
```
    результат каждого выражения запишите в переменную и выведите в консоль
    в комментариях рядом с каждым примером напишите развернуто, почему получился такой результат
